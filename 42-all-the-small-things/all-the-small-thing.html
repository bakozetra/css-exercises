`:focus` represents an element (such as a form input) that has received focus. It is generally triggered when the user clicks or taps on an element or selects it with the keyboard's "tab" key. We have to use focus on style.

:focus-within represents an element that has received focus or contains an element that has received focus. In other words, it represents an element that is itself matched by the :focus pseudo-class or has a descendant that is matched by :focus.

:visited represents links that the user has already visited. For privacy reasons, the styles that can be modified using this selector are very limited.

:targetrepresents a unique element (the target element) with an id matching the URL's fragment.

:disabled : pseudo-class represents any disabled element. An element is disabled if it can't be activated (selected, clicked on, typed into, etc.) or accept focus. The element also has an enabled state, in which it can be activated or accept focus.
